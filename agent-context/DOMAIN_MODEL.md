# TODP Domain Model

## Institution
An organization that provides one or more services.

Examples:
- Hospital
- School
- Bank
- Court
- Police Station

## Service
A capability offered by an institution.

Examples:
- Passport Renewal
- Medical Consultation
- Account Opening

## Location
A physical location where a service can be accessed.

## User
A person searching for a service.

## Search Query
A request made by a user to find a service.

## Discovery Hierarchy

Institution
↓
Local
↓
District
↓
State
↓
National
↓
International

## Ranking Factors

- Money Cost
- Time Cost
- Distance Cost

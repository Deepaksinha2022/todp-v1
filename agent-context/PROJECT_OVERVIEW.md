# TODP Project Overview

Mission:
Help users discover the right institution and service with minimum money, time, and distance cost.

Core Discovery Hierarchy:
1. Institution
2. Local
3. District
4. State
5. National
6. International

Primary Ranking Formula:
Money Cost
+
Time Cost
+
Distance Cost

Technology Stack:
- React
- FastAPI
- PostgreSQL
- pgvector
- Redis

Version:
V1

Current Goal:
Build and validate a working MVP that creates measurable user value.
